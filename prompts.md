# Prompt Engineering Documentation

## Project Name

AI Website Copy Generator for Local Businesses

## Business Name

SmileCare Dental Clinic

## Location

Visakhapatnam (Vizag), Andhra Pradesh, India

## Objective

Create professional, SEO-friendly website content for a fictional local dental clinic using structured AI prompts. The goal is to generate persuasive and user-friendly content that builds trust, highlights dental services, and encourages visitors to book an appointment.


---

## Prompt 1: Homepage Content Generation

### Objective

Generate engaging, SEO-friendly homepage content for SmileCare Dental Clinic that builds trust, highlights services, and encourages visitors to book an appointment.

### Prompt

You are an expert healthcare website copywriter and SEO content specialist.

Create professional homepage content for a dental clinic using the following details:

- Business Name: SmileCare Dental Clinic
- Location: Visakhapatnam (Vizag), Andhra Pradesh, India
- Target Audience: Families, children, working professionals, and senior citizens
- Services:
  - General Dentistry
  - Teeth Cleaning
  - Root Canal Treatment
  - Teeth Whitening
  - Dental Implants
  - Braces & Aligners
  - Pediatric Dentistry
  - Emergency Dental Care
- Brand Personality: Professional, friendly, trustworthy, modern, and affordable
- Goal: Build trust, explain services clearly, and encourage visitors to book an appointment.

Generate the following sections:

1. Hero Section
2. Headline
3. Subheadline
4. About Us
5. Why Choose Us
6. Featured Services
7. Call-to-Action (CTA)
8. SEO Meta Description
